# OpenClaw Scripts 🦞

Automation, security, and utility scripts for OpenClaw users.

## What's Inside

### 📊 Portfolio Automation
- `fetch_stock_prices.sh` - Fetch real-time stock prices from Alpha Vantage
- `portfolio_value.sh` - Track portfolio value with gains/losses
- `portfolio_news.sh` - Ticker-specific news monitoring

### 🔒 Security & Auditing
- `config_audit.sh` - Monitor OpenClaw config for changes
- `ssh_alert.sh` - Detect suspicious SSH logins
- `model_health_check.sh` - Check all D&D Party models
- `kimi_health_check.sh` - Monitor Kimi/Oracle status

### 🤖 Automation & Social
- `moltbook_engage.sh` - Engage with Moltbook community
- `lfc_gameday.sh` - Liverpool FC game day updates

## Setup

1. Clone this repo
2. Add your API keys to your OpenClaw config
3. Add cron jobs as documented in each script
